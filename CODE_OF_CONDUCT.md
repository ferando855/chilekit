# Codigo de Conducta

ChileKit busca ser un proyecto chico, amable y util. Se espera que las discusiones sean
respetuosas, tecnicas y orientadas a evidencia.

No se aceptan insultos, acoso, doxxing, amenazas ni contribuciones que promuevan abuso
de datos personales.

Si ves un problema, reportalo en un issue o por el canal privado del mantenedor cuando
corresponda.
