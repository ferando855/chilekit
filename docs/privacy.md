# Privacidad y Abuso de Datos

ChileKit no es un buscador de personas.

Permitido:

- Feriados, comunas, regiones, indicadores economicos y datasets publicos.
- Datos agregados, institucionales o estadisticos.
- Validacion local de formato y digito verificador de RUT.

No permitido:

- Rutificadores.
- Asociar RUT con nombre, domicilio, telefono, email personal u otra identidad.
- Scraping de perfiles personales.
- Reempaquetar filtraciones, padrones no autorizados o datos sensibles.

Si una fuente publica contiene datos personales por error o por mal diseño, ChileKit debe
filtrarla, degradarla o rechazarla antes de exponerla en CLI/MCP.
